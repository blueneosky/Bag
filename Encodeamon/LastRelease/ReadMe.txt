Copy content of file into an other file with specific encoding.
Usage :
  Encodeamon <encoding_out> <file_in> <file_out>
  Encodeamon -l
  Encodeamon -s <file>
  Encodeamon -h -help
    <encoding_out> : one of the available encoding
    <file>         : file to examine
    <file_in>      : file monitored and re-encode
    <file_out>     : file with result
    -l --list      : list available encoding
    -h --help      : this help
